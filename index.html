RADICAL-Pilot

RADICAL-Cybertools is a suite of tools designed to support science across HPC, Grids, and Clouds. RADICAL-SAGA provides users with a simple interface to interact with the different computing environments without worrying about the low-level details of each machine. RADICAL-Pilot builds upon RADICAL-SAGA to allow the users to coordinate the execution of a large number of tasks across many resources. Ensemble Toolkit builds upon RADICAL-Pilot to provide users with a simple way to express and execute scientific workflows. Taken together, the RADICAL-Cybertools suite gives the user the ability to execute scientific workflows without the burden of interacting with different machines or manage the execution of the individual tasks.
